# Missions and Game Mode

- Basic OPR missions and tournament packs can reward different list shapes. Match the army plan to the scoring environment when the user specifies one.
- Progressive scoring demands early board presence. Prioritize mobility, Scout, transports, enough activations, and units that can contest before the final round.
- In progressive scoring, Ambush can be risky because off-board units may not score or project pressure early enough.
- Basic objective scoring allows slower plans, shooting, Ambush, and late-game commitment, but Scout and mobility are still valuable.
- Age of Fantasy places extra value on mobility because melee, positioning, and scenario reach matter heavily.
- In Age of Fantasy, also check for Impact, banners or morale support, useful spells, and units that can fight for objectives after taking damage.
- Generalist lists are usually safer for competitive play. Skew lists can be strong, but they should include warnings about bad matchups and missing answers.
- For themed lists, preserve the theme when possible while naming the weaknesses that must be mitigated.
